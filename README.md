## Description

This repository contains the public disclosure version of the author’s working paper:

**N.E.X.U.S. — A Local Annihilation–Deconfinement Cosmogenesis Model: Core X, Dynamic Trapping Boundary and Observational Constraints**

The manuscript presents a theoretical research model concerning selected cosmological and astrophysical problems, including local cosmogenesis, matter–antimatter asymmetry, annihilation and deconfinement processes, dynamic trapping boundaries, neutrinos, Proca-sector contributions, old matter, post-annihilation remnants and observational falsification criteria.

This work is made publicly available for scientific reading, citation, criticism, verification and academic discussion.
